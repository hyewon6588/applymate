# ApplyMate

**Rate: Organized, Yours.**

ApplyMate is an AI-powered job application tracking tool that helps users manage job applications, evaluate resume-job match percentages, and receive keyword-based feedback to optimize their applications. Built with a modern tech stack, ApplyMate brings automation and insight to the job hunt process.

---

## Features

- **Application Tracker** ‚Äì Add and manage job applications with fields like company, position, location, and status.
- **File Uploads** ‚Äì Attach resume, job posting, cover letter, and transcript documents.
- **Match % Calculation** ‚Äì Automatically compute how well your resume aligns with a job posting.
- **Keyword Feedback** ‚Äì Identify missing keywords or skills that could improve your resume for a specific job.
- **AI-Driven Analysis** ‚Äì Powered by `sentence-transformers/all-mpnet-base-v2` for semantic similarity.
- **Storage** ‚Äì Uploaded files are securely stored via Supabase.
- **Full-Stack Web App** ‚Äì Built with FastAPI (backend) and Next.js (frontend), styled using Tailwind CSS.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/hyewon6588/applymate.git
cd applymate
```

### 2. Backend Setup

```bash
cd server
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Frontend Setup

```bash
cd client
npm install
npm run dev
```

### 4. Environment Variables

Create `.env` files for both frontend and backend. Include variables like:

#### `.env` (FastAPI)

```env
MONGO_URI=your_mongodb_connection_string
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_api_key
JWT_SECRET_KEY=your_secret_key
JWT_ALGORITHM=your_jwt_algorithm
```

#### `.env.production` (Next.js)

```env
NEXT_PUBLIC_API_URL=api_production_url

# === environment ===
NEXT_PUBLIC_ENV=production
```

## üß™ AI Matching Logic

Match percentages and keyword feedback are generated by comparing embeddings from the `all-mpnet-base-v2` model:

- **Match %**: Cosine similarity between job posting and resume.
- **Keyword Feedback**: Missing terms are extracted by analyzing term overlap and relevance.
